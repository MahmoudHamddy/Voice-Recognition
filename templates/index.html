Decision Tree: can be “learned” by splitting the source set into subsets based on an attribute value test. This process is repeated on each derived subset in a recursive manner called recursive partitioning. The recursion is completed when the subset at a node all has the same value of the target variable, or when splitting no longer adds value to the predictions.

Model features
Root-mean-square (RMS): It refers to the total magnitude of the signal
Zero crossing rate : It the rate at which a signal changes from positive to zero to negative or from negative to zero to positive.
Mel-Frequency Cepstral Coefficients (MFCCs): It is a representation of the short-term power spectrum of a sound, based on some transformation in a Mel-scale.
Chroma : It denotes to the characteristics of a certain pitch {C, C♯, D, D♯, E, F, F♯, G, G♯, A, A♯, B} is present in the sampled frame.
